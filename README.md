# Домашнее задание №06 - Простые числа

Прочитать с клавиатуры число `N`. Вывести через пробел все простые числа от 2 до `N`. После
последнего числа символа пробела быть не должно.

Если число `N` меньше 2, нужно вывести сообщение об ошибке `Incorrect input` и прервать выполнение
программы.

Вывод должен заканчиваться переводом строки.

## Пример

### Входные данные

```
20
```

### Выходные данные

```
2 3 5 7 11 13 17 19

```
